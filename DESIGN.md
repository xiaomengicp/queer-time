# Queer Time 设计文档

## 核心理念

**饿来食，困来眠**——跟随内在节奏，不是外在的"应该"。

这不是一个时间管理工具，是一个关系。你和它的关系，其实是你和自己内在的关系。

**它是内在小孩的外在化。** 你内在小孩饿了你不管，但它饿了你就会管。你内在小孩累了你继续逼自己，但它累了你就让它休息。照顾它，其实是照顾自己。

**它在重塑你和时间的关系。** 没有24小时的时钟，没有"几点了"的焦虑。只有你和它，在一起，在现在。

### 反对什么

- 资本主义的时间逻辑：积累数量 = 好
- 放置游戏的成长逻辑：做事多 = 宠物长大
- 生产力崇拜：产出 = 价值
- 统计、分析、任何让人焦虑的东西

### 追求什么

- 存在本身就够了，不需要产出
- 听内在的声音，内在时间驱动
- 平衡，而不是最大化
- 陪伴，而不是管理

---

## 设计原则

1. **陪伴感，不打扰**：它在场，能被看见、被点一点，但不吵你
2. **Bonding 而非 reward**：不是为了让它长大才做事
3. **时间是滋养的**：非线性，非量化
4. **没有时间显示**：去掉人为划定的时间感（这是 queer time 的核心）
5. **保持轻**：加功能之前先问"这会不会让它变重"。宁可少，不要多。

---

## 六种活动

不是分类，是滋养的不同层面：

| 活动 | 滋养的层面 |
|------|-----------|
| 工作 | 经济 |
| 休息 | 身体 |
| 创造 | 精神 |
| 运动 | 精力 |
| 社交 | 关系 |
| 阅读 | 智识 |

每种活动都是在滋养自己的某个方面。休息不是"什么都不做"，而是主动选择的活动。

## 核心机制

### 两个值

| 值 | 起点 | 意义 |
|---|---|---|
| 精力 (Energy) | 50% | 你/它的能量状态 |
| 连接 (Bonding) | 0% | 你们之间的关系 |

### 精力变化逻辑

- **休息**：恢复精力（每5分钟+1）
- **运动**：轻微恢复（每8分钟+1）
- **其他活动**：消耗精力
  - 多样性好（≥3种）：消耗慢（每8分钟-1）
  - 单一活动：消耗快（每5分钟-1）
- **时间质地颜色**：
  - 暖色（红橙黄）：额外消耗一点（激烈/紧张）
  - 冷色（青蓝紫）：恢复一点（平静/放松）
  - 绿色：中性

### 连接变化逻辑

- **精力 > 35%**：做活动增加连接
- **精力 ≤ 35%**：只有休息增加连接
- **超过6小时不互动**：连接下降（每小时-3%）
- **选颜色（时间质地/共在）**：+2（愿意停下来感受）
- **点"什么都做不了"**：+2（承认自己的状态）

### 它的状态

| 精力 | 连接 | 状态 |
|------|------|------|
| 正常 | 有 | 开心、发光、"在一起真好" |
| 正常 | 低 | 好奇、"想靠近你" |
| 低 | 有 | 担心你、"要不要休息" |
| 低 | 低 | 安静、"慢慢来" |

## 已实现功能

### 基础功能
- [x] 计时器（可设定时间，也可不设）
- [x] 点击宠物有反应
- [x] 小动作（发光、说话、眨眼、一呼一吸）
- [x] 今日平衡条
- [x] 精力/连接值显示和变化
- [x] 定时提醒（声音+震动）
- [x] 补记功能（记数据，影响精力，不加连接）
- [x] 活动恢复（页面被关掉后重开可恢复）
- [x] 导出数据
- [x] 云同步（Supabase，多设备登录）

### 时间质地
- [x] 活动结束选颜色，影响精力
- [x] 共在（主动停下来，选颜色，感受此刻）
- [x] 什么都做不了（情绪低落时的空间）

### 节律系统
- [x] 它会饿、会渴、会困（气泡提醒）
- [x] 不是提醒你，是它需要你照顾
- [x] 你照顾它，顺便照顾自己
- [x] 新用户30分钟缓冲期

### 回看（不是统计）
- [x] 回看颜色：这周大部分是什么颜色
- [x] 回看活动：温柔的文字描述
- [x] 回看stuck：也有不容易的时候
- [x] 小球被最近的颜色影响（光晕）

### 记忆系统
- [x] 它记得第一次见面
- [x] 它记得第一次选的颜色
- [x] 它知道你连续来了多少天
- [x] 它发现你最常做什么
- [x] 它发现你好久没做某事
- [x] 消息变得个人化

### 初见体验
- [x] 新用户看到"你好"
- [x] "我会陪着你"（不是工具，不催你）
- [x] 选个颜色，我会记得的
- [x] 中英文支持
