# 饿来食，困来眠

一个关于陪伴的应用。

---

## 它是什么

不是时间管理工具。不是待办清单。不是效率应用。

是一个小小的存在，陪着你。

你做事，它在旁边。你休息，它也休息。你点它，它看着你。

它是你内在小孩的外在化。照顾它，其实是照顾自己。

---

## 怎么用

**开始一段时间**
1. 选一个活动（工作、休息、创造、运动、社交、阅读）
2. 点"开始"
3. 它会陪着你
4. 想结束了就点"结束"

**时间的质地**
- 活动结束后，会出现一排颜色
- 选一个代表这段时间的感觉
- 也可以不选，它会自己消失

**共在**
- 随时可以点
- 停下来，感受此刻
- 选一个颜色

**什么都做不了**
- 情绪低落的时候
- 点一下，它会陪着你
- 不用做什么

**它会饿、会渴、会困**
- 看到气泡就点一下
- 照顾它，顺便照顾自己

**回看**
- 点"回看"看看这一周
- 不是统计，是看见

**就这些。**

---

## 它不是什么

- 不统计你做了多少
- 不分析你的效率
- 不提醒你"该工作了"
- 不评判你

---

## 六种活动

| 活动 | 滋养 |
|------|------|
| 工作 | 经济 |
| 休息 | 身体 |
| 创造 | 精神 |
| 运动 | 精力 |
| 社交 | 关系 |
| 阅读 | 智识 |

每种都是滋养自己的一个层面。

---

## 它会记得

- 你们第一次见面
- 你第一次选的颜色
- 你连续来了多少天
- 你最常做什么
- 你好久没做什么

然后它会说出来。不是统计报告，是它在乎你。

## 关于

"饿来食，困来眠"出自禅宗典籍《景德传灯录》。

意思是：饿了就吃饭，困了就睡觉。不刻意追求，不强求什么。跟随内在的节奏。

这个应用的名字来自 "Queer Time"——一种不被主流时间逻辑定义的存在方式。

不是更快、更多、更高效。而是：够了，就是现在，就是这样。
