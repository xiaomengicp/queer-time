# 饿来食，困来眠

一个关于陪伴的应用。

---

## 它是什么

不是时间管理工具。不是待办清单。不是效率应用。

是一个小小的存在，陪着你。

你做事，它在旁边。你休息，它也休息。你点它，它看着你。

它是你内在小孩的外在化。照顾它，其实是照顾自己。

---

## 怎么用

**开始一段时间**
1. 选一个活动（工作、休息、创造、运动、社交、阅读）
2. 点"开始"
3. 它会陪着你
4. 想结束了就点"结束"

**时间的质地**
- 活动结束后，会出现一排颜色
- 选一个代表这段时间的感觉
- 也可以不选，它会自己消失

**共在**
- 随时可以点
- 停下来，感受此刻
- 选一个颜色

**就这些。**

---

## 它不是什么

- 不统计你做了多少
- 不分析你的效率
- 不提醒你"该工作了"
- 不评判你

---

## 六种活动

| 活动 | 滋养 |
|------|------|
| 工作 | 经济 |
| 休息 | 身体 |
| 创造 | 精神 |
| 运动 | 精力 |
| 社交 | 关系 |
| 阅读 | 智识 |

每种都是滋养自己的一个层面。

## 关于

这个应用的名字来自 "Queer Time"——一种不被主流时间逻辑定义的存在方式。

不是更快、更多、更高效。而是：够了，就是现在，就是这样。
